# 🚀 思维导图导出功能 - 快速开始

## ✨ 新增功能

在现有思维导图AI问答功能的基础上，新增了**导出功能**：

- **📄 Markdown导出**：导出为.md文件
- **🖼️ PNG导出**：导出为高质量PNG图片

## 🛠️ 快速安装

### Windows用户
1. 双击运行 `start-export-features.bat`
2. 脚本会自动安装依赖并启动服务

### 手动安装
```bash
# 1. 安装后端依赖
cd backend
npm install puppeteer

# 2. 启动后端服务
npm start

# 3. 新开终端，启动前端服务
cd ../frontend
npm start
```

## 🎯 使用方法

1. **生成思维导图**：输入内容，点击"🚀 生成思维导图"
2. **导出Markdown**：点击绿色的"📄 Markdown"按钮
3. **导出PNG**：点击蓝色的"🖼️ PNG"按钮
4. **文件自动下载**：文件会保存到默认下载目录

## 🔧 技术特点

- **Markdown导出**：保持原有层级结构，支持版本控制
- **PNG导出**：高分辨率(1200px)，美观设计，自动调整高度
- **智能布局**：根据内容长度自动优化页面尺寸
- **美观设计**：渐变背景，彩色标题，圆角卡片

## 🚨 注意事项

- 首次PNG导出需要下载Chromium浏览器(~200MB)
- 确保后端服务正常运行
- 建议在性能较好的设备上使用PNG导出

## 💡 使用场景

- **Markdown**：编辑、协作、版本控制
- **PNG**：分享、演示、打印、社交媒体
