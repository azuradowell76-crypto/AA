## 🎯 使用方法

### 安装插件
1. 启动后端服务：`cd backend && npm start`
2. 打开Chrome扩展管理页面：`chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择`extension`文件夹

### 使用插件
- **点击图标**：点击浏览器工具栏中的🧠图标
- **快捷键**：按`Ctrl+Shift+M`
- **页面选择**：在任意网页选择文本，点击提示
- **浮动按钮**：点击页面右上角的🧠浮动按钮

5. **消息传递** → Background Script
